# dotfiles

## how to init.

1. **Clone to [user_vimrc] dir.**
    ```
    git clone [this_repo_url] dotfiles
    cd dotfiles
    ```
1. **You can easily install it using node.js.**
    ```
    node _install
    ```
    if not instaled node.js.
    ```
    # for linux(mac).
    cd _install && sh linux.sh

    # for windows.
    cd _install && windows.bat
    ```
